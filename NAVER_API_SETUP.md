# 네이버 책 검색 API 설정 가이드

책 검색 기능을 사용하기 위해 네이버 개발자 센터에서 API 키를 발급받아야 합니다.

## 1. 네이버 개발자 센터 접속

1. [네이버 개발자 센터](https://developers.naver.com/)에 접속합니다.
2. 네이버 계정으로 로그인합니다.

## 2. 애플리케이션 등록

1. **애플리케이션 등록** 메뉴를 클릭합니다.
2. 다음 정보를 입력합니다:
   - **애플리케이션 이름**: 예) "독서 활동 앱"
   - **사용 API**: "검색" 선택
   - **비로그인 오픈API 서비스 환경**: "웹" 선택
   - **서비스 URL**: `http://localhost:3000` (개발 환경)
   - **서비스 환경**: 
     - 개발 단계: `http://localhost:3000`만 입력해도 됩니다 (필수)
     - 배포 후: 운영 환경 URL을 추가로 등록하세요 (예: `https://your-domain.com`)

3. **등록하기**를 클릭합니다.

> **참고**: 개발 단계에서는 localhost URL만으로도 API를 사용할 수 있습니다. 나중에 배포할 때 운영 환경 URL을 추가로 등록하면 됩니다.

## 3. API 키 확인

1. **내 애플리케이션** 메뉴에서 등록한 애플리케이션을 선택합니다.
2. **Client ID**와 **Client Secret**을 복사합니다.

## 4. 환경 변수 설정

`.env.local` 파일에 다음을 추가합니다:

```env
NAVER_CLIENT_ID=발급받은_Client_ID
NAVER_CLIENT_SECRET=발급받은_Client_Secret
```

## 5. 개발 서버 재시작

환경 변수를 추가한 후 개발 서버를 재시작합니다:

```bash
npm run dev
```

## 참고사항

- 네이버 책 검색 API는 무료로 제공되지만, 일일 호출 제한이 있습니다.
- API 키가 설정되지 않은 경우, 책 검색 기능은 작동하지 않으며 수동 입력만 가능합니다.
- **개발 단계**: localhost URL만으로도 충분합니다.
- **배포 후**: 실제 도메인을 서비스 환경에 추가로 등록해야 합니다. (애플리케이션 수정 메뉴에서 추가 가능)

## 문제 해결

- **"책 검색에 실패했습니다" 오류**: API 키가 올바르게 설정되었는지 확인하세요.
- **검색 결과가 없음**: 다른 검색어로 시도하거나 수동 입력을 사용하세요.
- **CORS 오류**: 네이버 API는 서버 사이드에서만 호출 가능합니다. API Route를 통해 호출하도록 구현되어 있습니다.
